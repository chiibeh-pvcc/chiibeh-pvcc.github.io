Chapter 4 Terms & Definitions

Term Definition
LAN: Local Area Network
MAN: Metropolitan Area Network
WAN: Wide Area Network
BUS Network: Topology in which all machines are connected to a common communication line, called a BUS.
Star Network: Topology where all machines are connected to a central Switch/HUB
CSMA/CD Ethernet: Carrier Sense, Multiple Access w/ Collision Detection: Protocols that dictates that each message be broadcast to all the machines on the bus.
CSMA/CA: Carrier Sense, Multiple Access with Collision Avoidance: Protocol where instead of detection of collisions, avoidance is paramount. Gives advantage to machines that have already been waiting to communicate.
Repeater: A device that passes signals 2 Buses without consideration for the meaning of the signals. Usaully with some form of amplification.
Bridge: Similar to but more complex than a repeater. Smart appliance that looks at destination address that accompanies each message and forwards a message across the connection only when that message is destined for a computer on the other side.
Switch: Is essentially a bridge with multiple connections. Allowing it to connect multiple Buses
Router: Special purpose computers that forward messages. It allows each network to keep its own internal characteristics.
Gateway: The "point" at which one network is llinked to the internet.
P2P/peer-to-peer: The client/server model involves one process (the server) providing a service to numerous others (clients) Provides and receives service from others.
Distributed Systems: Software units that woks as processes on different systems.
Cloud Computing: Huge pools of shared computers on the network can be allocated for use by clients as needed
Tier-1: ISPs that consist of very high-speed, high-capacity, international WANs. These networks are thought of as the backbone of the Internet. T
Tier-2: ISPs that tend to be more regional in scope and less potent in their capabilities.
Tier-3: An access ISP is essentially an independent internet, sometimes called an intranet, operated by a single authority.
IP Address: ICANN ICANN.ORG
DNS: Domain Name System- Used as an internet-wide directory system. Converts IP to mnemonic and vice-versa
TLD Top Level Domain: Used to reflect a domains classification. (.com for companies,.edu for education,etc..)
Country Code TLDS: Worldstandards.eu
Internet Protocols: FTP File Transfer Protocol
HTTP: Hyper Text Transfer Protocol
SMTP: Simple Mail Transfer Protocol defines a way that two computers on a network may inteeract when transmitting an emailmessage fro one host to the other.
URL: Uniform Resource Locator
Client-Side Activities: Additional funtioality to a website or an application that are performed by a client such as a browser.
Server-side: Activities performed by the server. Such as a webserver
Application Layer: Layer from which messages generally originate. Consists of software units, clients and servers that use the internet to completer their tasks.
Transport Layer: Adds tags that denote sequence of packets to be reassembled in order.
Network Layer: Determines the direction in which packets should be sent. Generally comprised of software on residing on routers.
Link Layer: Responsible for tranferring packets. Must determine the protocol for communication.
The Internet is: A packet switching network
Fault tolerant: Property of a system that enables it to continue operating properly even in the event of failure.
Redundant: Having multiple paths to access data routes, to increase reliability.
Cybersecurity: The study of protecting computers, networks, and data from such attacks.
Malware: Malicious Software
Virus: Software that infects a computer by inserting itself into programs that already reside in the machine. Executes, when the host program does.
Worm: An autonomous program that transfers itself through a network, taking up residence in computers and forwarding copies of itslef to other computers.
Trojan Horse: Program that enters a computer system disguised as a desirable program.
Spyware: Software that collects information about activities at the computer on which it resides and reports back to the insigator.
Phishing: Technique of obtaining information by simply asking for it. By pretending to be some trusted or authoritative entitiy.
DDoS Attack: Distributed Denial of Service attack. DoS attacks required a high volume of messages in a short time, to overwhelm the targeted server, but using a Botnet, the attack can be executed by many hosts.
Firewall: Filters traffic passing through a point in the network.
SPAM Filter: Firewall designed to block unwanted email
Proxy Server: Software unit that act as intermediary between a client and server, with goal of shielding client from adverse actions of server.
Antivirus Software: Software used to detect and remove the presence of known viruses and infections.
Cryptography: Study of sending and receiving messages securely in the presence of adversaries.
Public Key Encryption Used to encrypt messages that can only be decrypted by the private key. Public key is freely distributed, private key is held in safety known only to the end user.
