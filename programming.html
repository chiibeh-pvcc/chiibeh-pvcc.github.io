Chapter 5: Programming Algorithms & Software Testing

Phases of Computer Development: Phases 1-7
Phase 1: Understand the Problem
Phase 2: Design an algorithm to solve the problem
Phase 3: Write the program code in appropriate language based on the algorithm
Phase 4: Enter the program and save
Phase 5: Execute the program
Phse 6: Test the program, find errors and debug, then retest
Phase 7: Document and evaluate the program
Programming Structures: Terms
Sequential Structures Top-down execution
Decision structures: if/then/else
Iterative (Looping):
  • While loop: Continue looping as long as something is true.
  • Repeat/until loop: Continue looping until something is true.
  • For loop: Loop a given number of times.

Recursive Structures:

Two Classic Programming/Database Problems
1: Sorting
  • Descending Order/Ascending Order
  • Sorting Numbers
  • Sorting Strings (In accordance with ASCII values)
2: Searching
  • Finding specific data that meet certain criteria
  • In database
    1. Filters: Temporary Search
    2. Queries: Search criteria are saved so they don't have to be recreated
Two Popular search Techniques
1: Sequential Search Search from top to bottom or bottom to top, one item at a time
2: Binary Search Only works if the data is sorted first, start in the middle then go to half of that, etc
Two Classic Program Goals
1: Efficiency Program takes minimal resources and/or time
2: Correctness Program runs as expected fro all expected conditions
