Chapter 3 Terms & Definitions

Term Definition
1: Operating System The software that controls the overall operation of the computer including file management, the user interface, and the program execution environment.
2: Job queue A queue is a storage organization in which objects (in this case, jobs) are ordered in first-in, first-out (abbreviated FIFO and pronounced “FI-foe”) fashion
3: FIFO Objects are removed from the queue in the order in which they arrived.
4: Interactive processing Allows a program being executed to carry on a dialogue with the user through remote terminals.
5: Time-sharing Operating systems that provide service to multiple users at the same time.
6: Multiprogramming Divides time into intervals and restricts the each job to only one interval at a time.
7: Application Software Consists of the programs for performing tasks particular to the machine's utilization.
8: System Software Provides the system infrastructure that the application software requires. Performs the common computer system tasks.
9: Utility software Consists of software units that extend the capabilties of the operating system.
10: Graphical User Interface Represents files and programs pictorially as icons to be manipulated.
11: Shell Older user interface that uses textual messages on a keyboard and monitor to manipulate files.
12: Kernel Internal part of an Operating system, that performs the very basic functions required by the computer installation.
13: Device Driver The software units that communicate with the controllers (or at times, directly with peripheral devices) to carry out operations on the peripheral devices attached to the machine
14: File manager Coordinates the use of the machine's mass storage facilities.
15: Memory manager Coordinates the machine's use of main memory.
16: Virtual memory Fictional memory space created by paging.
17: Boot strapping Procedure that transfers the operating system from mass storage to main memory.
18: ROM Special nonvolatile memory cells that are designated as Read-only memory. Store boot loader.
19: Deadlock A condition in which two or more processes are blocked from progressing because each is waiting for resources reallocated elsewhere.
20: Spooling Holds data for output a more convenient time.
21: Super user or Administrator Person(s) with privileged access to the operating system.
22: Multi-factor authentication (MFA) Authentication method that require more than one piece of evidence to gain access.
